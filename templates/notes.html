Notes and Fixes

Here is a collection of short notes for various tasks and fixes to arbitrary issues I encounter. Although they're here mainly for my future self, they may also be of use to others looking for similar information.

Nexus 4: no themes are available for the Themer application

The list of themes (in any category - recent, popular, etc.) is empty on the Nexus 4.

This is due to a custom ROM altering the DPI of the UI. Themer is only compatible with the stock DPI, so to sort this problem, simply revert your ROM back to the default DPI. This point is actually mentioned on the app's Play Store entry. I should really read more...

Using TeXcount to provide a wordcount in complex LaTeX documents

Download TeXcount and install the Perl script to somewhere like /usr/bin/. Run texcount with the -merge option on the main TeX file to see word counts on the complete document.

NOTE use \include{} rather than \input{} in the main TeX file to prevent file-not-found errors.

Resolution problems with Trine 2 on Linux

This problem seems to occur when running the Steam version of Trine 2 on GNU/Linux systems with nVIDIA cards running with multiple heads. The issue is that the game tries to set the resolution to the maximum width and height of all the heads combined, meaning that the picture is stretched and only partially visible.

To fix the issue, edit the file trine2_bin_starter.sh in the relevant place in the Steam directory, and set the FB_FULLSCREEN_WIDTH and FB_FULLSCREEN_HEIGHT to your desired values (e.g. 1920x1200).

Issues typesetting TeX files using texlive on Arch Linux

This problem prevents the typesetting of any documents, returning a fatal error regarding the file /var/lib/texmf/web2c/pdftex/pdflatex.fmt. This seems to be a problem with pacman.

This can be fixed using the bash script contributed by darkdigitaldream on the Arch Linux forum.
