CasaStream

Current version: 1.0

Project home

Python, music streaming, PulseAudio

A system designed to support zoned music streaming around the home. In particular, allows music playing from one 'Master' server to be played simultaneously by other networked CasaStream servers. Once running, a user can choose which sound sources (applications) to stream and which devices should receive and playback the stream.

Whilst effectively just a wrapper for PulseAudio and VLC, it is designed to be mostly zeroconf, self-organising, self-detecting and easy to deploy.

WekaPy

Current version: 1.2

Download current version

Project home

Python, machine learning

A Python wrapper for the Weka machine-learning toolkit. Currently supports training a variety of different classifier types and testing (making predictions) against trained models. The library supports use of ARFF files or more programmatic use (as demonstrated in the repo's README). Direct to S3 File Uploads in Python

Heroku

7th May 2013

View article

Heroku, Python, JavaScript, Amazon S3

An article for Heroku's Dev Center covering a procedure enabling direct-to-S3 uploading using Python running on a signing server. The motivation of the article was to encourage Heroku (and any other) developers to make uploads to Amazon's S3 service, but bypassing any server-side processing. By simply using the server to sign the request and then having JavaScript do the actual legwork in uploading the files, the server is not tied up in processing the file and it will be able to continue responding to any requests made during the upload.
