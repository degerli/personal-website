Direct to S3 File Uploads in Python

Heroku

7th May 2013

View article

Heroku, Python, JavaScript, Amazon S3

An article for Heroku's Dev Center covering a procedure enabling direct-to-S3 uploading using Python running on a signing server. The motivation of the article was to encourage Heroku (and any other) developers to make uploads to Amazon's S3 service, but bypassing any server-side processing. By simply using the server to sign the request and then having JavaScript do the actual legwork in uploading the files, the server is not tied up in processing the file and it will be able to continue responding to any requests made during the upload.
