Gower Tides

Current version: 1.4

View source

View on Google Play

Android, Java, surf API, weather API, visualisation

An Android app for displaying tidal and weather data for the Gower Peninsula in South Wales. Differentiates from other similar tide applications since the calculations are made offline, which is useful due to limited data services in some areas of the Gower.

Also displays the surf data for a selection of four Gower locations.

CasaStream

Current version: 1.0

Project home

Python, music streaming, PulseAudio

A system designed to support zoned music streaming around the home. In particular, allows music playing from one 'Master' server to be played simultaneously by other networked CasaStream servers. Once running, a user can choose which sound sources (applications) to stream and which devices should receive and playback the stream.

Whilst effectively just a wrapper for PulseAudio and VLC, it is designed to be mostly zeroconf, self-organising, self-detecting and easy to deploy.

WekaPy

Current version: 1.2

Download current version

View source

Python, machine learning

A Python wrapper for the Weka machine-learning toolkit. Currently supports training a variety of different classifier types and testing (making predictions) against trained models. The library supports use of ARFF files or more programmatic use (as demonstrated in the repo's README). Provides a set of useful classes for handling instances and for getting useful information out of the predictions.

Minecraft Server Manager

Current version 2.0

This project is no longer under development (last update 2011).

Download current version

Java, Minecraft

A UI-based server manager for Minecraft for OS X. Designed to transparently and easily handle a Minecraft server running on a Mac, supporting functionality for creating and handling worlds, interacting with players (bans, OPs, etc.), issuing server commands and managing plugins for the Buckit server.

While base functionality still mostly works (creating worlds, updating the server, starting/stopping the server/server commands, etc.), a lot of the player interaction stuff relied on previous versions of the server's output and so will no longer work.
