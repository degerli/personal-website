Fixes

Here is a collection of fixes to arbitrary issues I encounter. Although they're here mainly to help my future self, they may also be of use to others with similar problems. Mostly these are issues that can't always be solved with simple searches.

Resolution problems with Trine 2 on Linux

This problem seems to occur when running the Steam version of Trine 2 on GNU/Linux systems with nVIDIA cards running with multiple heads. The issue is that the game tries to set the resolution to the maximum width and height of all the heads combined, meaning that the picture is stretched and only partially visible.

To fix the issue, edit the file trine2_bin_starter.sh in the relevant place in the Steam directory, and set the FB_FULLSCREEN_WIDTH and FB_FULLSCREEN_HEIGHT to your desired values (e.g. 1920x1200).

Issues typesetting TeX files using texlive on Arch Linux

This problem prevents the typesetting of any documents, returning a fatal error regarding the file /var/lib/texmf/web2c/pdftex/pdflatex.fmt. This seems to be a problem with pacman.

This can be fixed using the bash script contributed by darkdigitaldream on the Arch Linux forum.
