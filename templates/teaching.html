Supervised Projects

Undergraduate projects supervised during my PhD from September 2010 to 2013.

Social Media Visualisation Toolkit

Two projects of the same title were supervised:
Produce a design and a prototype system to generate interesting and useful visualisations to represent social media data for use with COSMOS's system. The system, written primarily in PHP and JavaScript, could accept data in order to produce dynamic visualisations, utilising an array of toolkits and APIs (D3.js, Google Charts, Google Maps, etc.). The system included functionality allowing account creation, the saving of visualisations and their source data and some further configuration (differing between the two supervised projects).

Digitial Asset Bank

Produce a design and a prototype system representing a digital asset bank. Users are able to upload various types of electronic media (documents, images, movies, 3-D models, etc.), which would then be made available for other users to purchase and download. The system incorporated various algorithms for detcting popular assets and allowed for appropriate previews of assets before purchases are made.

Coursework Assistance Tool

Produce a design and a prototype system to help students in managing and tracking their various University courseworks. The system was produced in PHP and enabled users to view upcoming deadlines, create and manage milestones, track and keep up-to-date with other group members, submit questions for staff, and so on.

Online Library System

Produce a design for an online system for a local library so that users can access eBooks, audiobooks, music, movies, etc online using their library account. This design was extended to work with mobile clients (Android, iOS, Kindle) and encompassed the legal aspects (copyright, etc).

Geo-Information Sharing for Mobile Devices

Produce a design for a system for smartphones (particularly Android, but extended to iOS) that would allow users to share geographical information on geographical features (buildings, points of interest, roads, natural features, etc) via a map-based positioning system. The design included how GPS would be used to locate the features, database design, how data would be shared & updated, and so on.
